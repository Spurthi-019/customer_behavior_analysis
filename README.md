# Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from data extraction and exploration to dashboard creation and reporting. The objective was to analyze the dataset, uncover meaningful insights, and present the findings through interactive visualizations and business reports.

The project includes:

* Data loading and preprocessing using Python
* Exploratory Data Analysis (EDA)
* Data cleaning and transformation
* SQL analysis using PostgreSQL
* Interactive dashboard development in Power BI
* Insight reporting and presentation creation using Gamma

---

## Dataset

The dataset contains business-related information used for analytical purposes. It includes multiple attributes that were explored to identify trends, patterns, and key performance indicators (KPIs).

### Dataset Activities

* Loaded dataset using Python
* Examined data structure and data types
* Identified missing values and duplicates
* Performed data validation checks
* Prepared data for analysis and visualization

## Data Analysis Process

### 1. Data Loading

* Imported the dataset into Python.
* Verified schema, columns, and data quality.
* Conducted initial inspection of records.

### 2. Exploratory Data Analysis (EDA)

* Generated summary statistics.
* Analyzed distributions and trends.
* Identified outliers and anomalies.
* Explored relationships between variables using visualizations.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Standardized formats and column names.
* Corrected inconsistent data entries.
* Prepared a clean dataset for further analysis.

### 4. SQL Analysis

* Loaded cleaned data into PostgreSQL.
* Created database tables.
* Executed SQL queries to answer business questions.
* Used filtering, aggregation, grouping, and joins to derive insights.

Example analyses:

* Top-performing categories
* Trend analysis
* KPI calculations
* Comparative performance metrics

## How to Run the Project

### Prerequisites

Install the required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn psycopg2
```

### Steps

1. Clone the repository.

```bash
git clone <repository-link>
```

2. Open the project folder.

```bash
cd project-folder
```

3. Run the Python scripts for data loading, cleaning, and EDA.

```bash
python main.py
```

4. Load the cleaned dataset into PostgreSQL.

5. Execute the SQL scripts provided in the `sql` folder.

6. Open the Power BI dashboard file (`.pbix`) to explore visualizations.

7. Review the project report and presentation for detailed findings.
